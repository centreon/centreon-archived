=======================
Architectures possibles
=======================

Centreon permet plusieurs choix dans la composition de l'architecture de votre outil de supervision. D'une architecture relativement simple avec un serveur hébergeant tous les services, l'architecture peut aussi être organisée autour d'un découpage stratégique permettant de répartir la charge de collecte sur plusieurs serveur avec la mise en place de points de collecte sur plusieurs contninents.

Vous trouverez ici toutes les architectures supportées par cette version 2.8 de l'interface.
