#################################
Comment déployer la supervision ? 
#################################

Avec Centreon vous avez la possibilité de superviser des environnements relativement variés. Serveurs, application, UPS, sites web, équipements réseau : tous ces environnements ont leur spécificités. C'est pour cette raison que Centreon met à disposition de ses utilisateurs des "Plugins Packs" basés sur les centreon plugins. Pour vous aider à implémenter facilement ces derniers au sein de votre Centreon, cette documentation a pour vocation de vous aider à configurer la supervision de ces environnements dans Centreon.

.. toctree::
   :maxdepth: 1
   
   sso
   
Si vous avez des expériences que vous souhaitez partager avec la communauté Centreon, n'hésitez pas à vous en faire part sur le github de Centreon. Votre guide pourrait faire partie de la documentation officielle de Centreon.
